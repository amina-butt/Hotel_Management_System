# Hotel Management System

This is a simple Hotel Management System built for academic purposes. It provides basic functionality to manage room reservations, customer details, billing, and check-in/check-out operations.

## 📁 Project Structure

The project contains the following components:

- **Source Code**: All logic for room booking, customer handling, and hotel operations.
- **User Interface**: Simple text-based interface for interaction.
- **Executable/Main File**: The main file to run the system (usually named `main.cpp`, `hotel.cpp`, or similar).

## 🚀 Features

- Add new customers
- Reserve rooms
- Check in / Check out
- Display room availability
- Generate bills

## 🛠️ Technologies Used

- C++
- Basic file handling and procedural programming

## 📦 How to Run

1. Open the project folder.
2. Compile the main source file using a C++ compiler like `g++`:
   ```bash
   g++ hotel.cpp -o hotel
